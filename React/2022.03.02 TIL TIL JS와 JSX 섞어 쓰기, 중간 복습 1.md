## JS와 JSX 섞어 쓰기

```jsx
const Number = ({ number }) => {
      return <>{number % 2 === 0 ? <h1>{number}</h1> : <h5>{number}</h5>}</>;
    };

    const element = (
      <>
        <Number number={1} />
        <Number number={2} />
        <Number number={3} />
      </>
    );
```

이미 JS와 JSX를 섞어 사용하고 있어요!

HTML에서도 이미 쓰고 있어요!
